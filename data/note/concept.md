# Concept Note

This is a **sample markdown note** to demonstrate the note viewer functionality.

## Features

The note viewer supports:

- **Markdown rendering** with full CommonMark support
- **URL-based loading** - add `?note=filename` to the URL
- **Automatic title extraction** from the first heading
- **Read time estimation**
- **Responsive design** with the toolbox's visual theme
- **Syntax highlighting** for code blocks
