🪟 Window Subdivision λ·Poisson

From static total count → independent windows → infinite path measure

📐 Why windows define probability

A Poisson process with rate λ has no natural “density” on exact event times. Instead, the probability measure is built from finite-dimensional windows.

🧠 Core idea: For any disjoint intervals I₁, I₂, … the counts N(I₁), N(I₂), … are independent Poisson with mean λ·|I|.
➜ This fully determines the infinite‑path behavior (Kolmogorov extension).

Subdivision reveals structure:
• 1 window → only total count (static regime).
• Many windows → local rate fluctuations become visible.
• Limit of infinitesimal windows → each interval of length dt behaves like a biased coin (prob λ·dt of a click).
