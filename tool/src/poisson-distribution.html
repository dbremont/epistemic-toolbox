Poisson distribution X ~ Poi(λ)

the fundamental atom of counts · independent increments · rare events λ = mean = variance

⚛️ Why Poisson anchors everything

The Poisson distribution describes the number of events occurring in a fixed interval when these events happen independently at a constant average rate λ.

P(X = k) = e–λ · λk / k!
k ∈ {0, 1, 2, … } · λ > 0

This single discrete distribution is the building block of the Poisson process. Every window count in a homogeneous Poisson process follows exactly this distribution.

✨ Deep properties (why it's the atom)

  • ① Mean = Variance = λ — unique among common distributions.
  • ② Sum of Poissons : X₁~Poi(λ₁) + X₂~Poi(λ₂) = Poi(λ₁+λ₂) (independent).
  • ③ Thinning : randomly keeping each event with prob p yields Poi(p·λ).
  • ④ Rare event limit : Binomial(n, p) → Poisson(λ) when n→∞, p→0, np→λ.

“To know the Poisson process, you must first befriend the Poisson distribution.”
