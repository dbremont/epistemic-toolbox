Understanding statistical variance

What is Standard Deviation?

Standard deviation (σ) measures how spread out data points are from the mean (μ). A low standard deviation means data points are close to the mean, while a high standard deviation indicates data is spread over a wider range.

Formula: σ = √( Σ(xi - μ)² / N )

Where μ is the mean, xi are individual data points, and N is the count.

Using the Tool

Add Data Points: Enter values manually using the input field, quick-add buttons (+10, +25, etc.), or click directly on the canvas to add points.

Load Presets: Use the "Data" button to load preset distributions:

  • Normal: Bell-shaped distribution, most common in nature
  • Uniform: Evenly distributed values
  • With Outliers: Normal distribution with extreme values
  • Bimodal: Two distinct peaks

Score Modes: Switch between Z-Score and T-Score using the mode selector. T-Score is normalized to mean=50, std=10.

Visualization Layers: Toggle sigma bands (±1σ, ±2σ, ±3σ) and normal curve overlay to understand data distribution.

Reset: Use Reset button to return to initial state.

Refresh: Use Refresh button to regenerate current data.

Understanding the Results

Statistics Panel: Displays mean (μ), standard deviation (σ), variance (σ²), and range of your data.

Visualization: Shows data points on a timeline with colored sigma bands indicating confidence intervals.

Score Interpretation: Each data point is categorized:

  • Common (|Z| ≤ 1): Within one standard deviation (68% of normal distribution)
  • Uncommon (1 < |Z| ≤ 2): Between one and two standard deviations (27% of normal distribution)
  • Rare / Outlier (|Z| > 2): Beyond two standard deviations (5% of normal distribution)

Score Calculator: Enter any value to calculate its Z/T score and percentile rank.
