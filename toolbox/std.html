Understanding Standard Deviation

The Formulas

Standard Deviation:

σ = √( Σ(xi - μ)² / N )

Z-Score:

Z = (x - μ) / σ

μ = mean, xi = value, N = count, x = data point

What Z-Score Tells You

Z-score standardizes any data point by measuring how many standard deviations it is from the mean. This lets you compare values from different distributions.

● Z = 0: Exactly at the mean (average)

● Z = +1: One standard deviation above mean

● Z = -2: Two standard deviations below mean
