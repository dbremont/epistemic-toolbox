# Tags

> Standardized semantics for tag usage in the epistemic framework.

* `#activity` — denotes an **epistemic activity** (e.g., observing, modeling, inferring)
* `@artifact` — denotes an **epistemic artifact** (e.g., model, dataset, proof)
* `$meta` — denotes **meta-epistemic annotations** (i.e., statements about epistemology itself)
* `*class` — denotes a **class or category of epistemic tools**
*  others  tags.

Rules:

- Differentiate tag types using a distinct color for each category of tag (e.g., #activity, @artifact, $meta, class), ensuring consistent visual encoding across the system.
- Update the help modal accordingly.
